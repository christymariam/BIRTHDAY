# 🎉 Happy Birthday Caroline! 🎂

A beautiful, interactive web-based birthday surprise with animations, audio effects, and vivid colors!

## 🚀 Quick Deploy (Easiest Way)

### Option 1: One-Click Deploy to GitHub Pages

1. **Create a GitHub account** (free)
2. **Create a new repository** called `birthday-surprise`
3. **Upload all files** from this folder to your repository
4. **Go to Settings → Pages**
5. **Select "main" branch** and save
6. **Your link will be:** `https://YOUR_USERNAME.github.io/birthday-surprise/`

### Option 2: Automated Deploy Script

**For Windows:**
```bash
# Double-click deploy.bat or run in Command Prompt:
deploy.bat
```

**For Mac/Linux:**
```bash
# Make script executable and run:
chmod +x deploy.sh
./deploy.sh
```

### Option 3: Instant Deploy to Netlify

1. Go to [netlify.com](https://netlify.com)
2. Drag and drop the `birthday.html` file
3. Get your instant link!

## 🎨 Features

- **7 Interactive Pages**: Welcome, Note, Cake, Animation, Fortune, Memories, Personal Message
- **Smooth Animations**: Floating balloons, flickering candles, growing numbers
- **Audio Effects**: Safe Web Audio API sounds for clicks and transitions
- **Vivid Colors**: Rainbow gradients and bright, cheerful design
- **Responsive**: Works on phones, tablets, and computers
- **No Installation**: Just click the link!

## 🛠️ Manual Setup (If needed)

If automated deploy doesn't work:

1. **Install Node.js** (if not installed)
2. **Run locally**: `npm start`
3. **Deploy**: `npm run deploy`
